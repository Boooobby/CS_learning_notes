## U04

### 重要知识点

1. 受控接入：轮询 令牌传递
2. 随机接入：
   1. 纯ALOHA
   2. 时隙ALOHA
   3. CSMA
   4. CSMA/CD
3. ALOHA：想发就发 脆弱期2T（T为帧发送时间
4. 时隙ALOHA：脆弱期T
5. CSMA：先听后发 
   1. 非坚持 1坚持 p坚持
   2. 坚持——忙就一直听 非坚持——忙就过会听
6. CSMA/CD 先听后发 边发边听
   1. 二元指数后退：经历n次冲突 从0 - 2^n - 1中选（n小于等于m
   2. 争用期：Slot time = 2T = 2S / (0.7C) + 2t_phy
   3. 半双工（要是是全双工就不会有冲突）
   4. L_min = Slot Time × R 最小帧长
7. CDMA
   1. 1个bit变成m位的序列 +1 -1
   2. 内积运算 正交
   3. （+1 -- 1） （-1 -- 0）

### 名词解释

1. CSMA -- 载波监听多点接入
2. CD -- 冲突检测
3. TDM FDM WDM -- 时分复用 频分复用 波分复用
4. STDM -- 统计时分复用
5. DWDM -- 密集波分复用
6. CDMA -- 码分复用
