这份 **Tmux 科研“生存”速查表** 是为你量身定制的。我把它们分为了 **“必须背下来的”** 和 **“让你效率翻倍的”** 两类。

### 核心概念：前缀键 (The Magic Key)

所有的快捷键（Shortcuts）都需要先按这个组合：
👉 **`Ctrl + B`**
*(下文统称为 **`Prefix`**。意思是：按住 Ctrl 不放，按一下 b，**两手完全松开**，再按后续的键)*

---

### 第一部分：命令行操作 (在进入 Tmux 之前)

*这些命令是在你的 iTerm2 普通命令行里敲的。*

| 动作 | 命令 | 解释 |
| --- | --- | --- |
| **新建会话** | `tmux new -s <名字>` | **最常用**。例如：`tmux new -s train_v1`。一定要起名，不然以后找不到。 |
| **查看列表** | `tmux ls` | 看看后台现在有几个“地下室”开着。 |
| **重连会话** | `tmux attach -t <名字>` | 回到指定的会话。简写：`tmux a -t <名字>`。 |
| **杀死会话** | `tmux kill-session -t <名字>` | 彻底关掉某个任务（不仅仅是断开连接，而是从内存清除）。 |
| **全部杀死** | `tmux kill-server` | **慎用**。关闭服务器上属于你的所有 Tmux 会话。 |

---

### 第二部分：会话与分屏 (进入 Tmux 之后)

*这些是快捷键，先按 `Prefix` (Ctrl+b)，松手，再按后面的键。*

#### 1. 保命操作 (最重要)

| 动作 | 快捷键 | 解释 |
| --- | --- | --- |
| **挂起/离开** | **`Prefix`** 然后 **`d`** | **Detach**。人走，程序留。回到普通命令行。 |
| **重命名会话** | **`Prefix`** 然后 **`$`** | 假如你创建时忘了起名，可以用这个改名。 |

#### 2. 分屏操作 (一边跑代码，一边看显卡)

| 动作 | 快捷键 | 解释 |
| --- | --- | --- |
| **左右切分** | **`Prefix`** 然后 **`%`** | 屏幕一分为二（左右）。需按 `Shift + 5`。 |
| **上下切分** | **`Prefix`** 然后 **`"`** | 屏幕一分为二（上下）。需按 `Shift + '`。 |
| **切换光标** | **`Prefix`** 然后 **`方向键`** | 在不同的格子之间跳来跳去。 |
| **关闭当前格** | **`Prefix`** 然后 **`x`** | 会问你 `kill-pane? (y/n)`，输 `y` 回车。或者直接输 `exit` 命令也行。 |
| **放大/缩小** | **`Prefix`** 然后 **`z`** | **神器**。把当前小格子全屏（Zoom），方便看报错日志，再按一次还原。 |

#### 3. 翻页与查看历史

*默认情况下鼠标滚轮在 tmux 里不能翻历史，除非你配置了鼠标支持。如果没有配置，用下面的方法：*

| 动作 | 快捷键 | 解释 |
| --- | --- | --- |
| **进入复制模式** | **`Prefix`** 然后 **`[`** | 进入后，屏幕右上角会显示页码。**这时候可以用方向键或 PageUp/Down 翻看以前的日志**。 |
| **退出查看** | **`q`** | Quit。看完日志按 q 回到正常模式。 |

### 💡 给科研新手的 3 个最佳实践

1. **一定要给 Session 起名字**：
* ❌ 不要只输 `tmux`（系统会叫它 0, 1, 2... 几天后你就忘了哪个跑的是哪个模型了）。
* ✅ 要输 `tmux new -s diffusion_test` 或 `tmux new -s resnet_L20`。


2. **善用 Zoom (`Prefix + z`)**：
* 当你把屏幕切得碎碎的（左边代码，右边显卡），突然代码报错了，密密麻麻看不清。
* 按一下 `Prefix + z`，代码窗口瞬间全屏，看清楚错误后再按一次缩回去，非常优雅。


3. **配置一次鼠标支持（强推）**：
* 虽然键盘流很酷，但初学者能用鼠标滚轮翻页真的很重要。
* 在服务器运行一次这个命令（永久生效）：
```bash
echo "set -g mouse on" >> ~/.tmux.conf && tmux source ~/.tmux.conf

```


* 以后你就可以直接**用鼠标点击**来切换分屏，或者**滚动滚轮**看历史日志了。
